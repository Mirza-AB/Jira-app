# Jira Clone

A full-stack Jira-inspired ticketing application built with Spring Boot (backend) and React (frontend).  

## Overview
It supports project workflows, role-based permissions, tickets, comments, and JWT auth with refresh tokens.


## Live Demo
- Backend API (EC2): http://13.59.131.88:8080
- Frontend: Going live soon.


## Tech Stack

### Backend
- Java 17
- Spring Boot 3.1.4
- Spring Data JPA (Hibernate)
- PostgreSQL
- Spring Security with JWT
- Swagger for API docs
- Maven + Docker

### Frontend
- React (in development)
